# Text2SQLi

# Data
Data is taken from Kaggle: https://www.kaggle.com/datasets/arnavsmayan/netflix-userbase-dataset
Added additional columns such as Password, Salt and Username generated randomly.
Password column was generated using random string + salt string hashed using SHA256.
Salt and Usernames are generated using random strings.
A total of 2.5k rows of data.
